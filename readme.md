Opensource PCB for bully keyboard
------------------------------

This is an opensource pcb made for [Bully keyboard](https://mkh.works/#bully).

### Features
- No big bar
- MX/Alps
- Capslock LED
- EC11 encoder
- RP2040

### Production files
